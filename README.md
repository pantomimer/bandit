# OverTheWire: Bandit - 攻略記録

## 概要

OverTheWire の Bandit は、Linux コマンドラインおよびセキュリティの基礎を学ぶための入門 Wargame。
Level 0 から順番にパスワードを入手し、次のレベルへ進む形式。

URL: https://overthewire.org/wargames/bandit/

## 接続情報

- ホスト: bandit.labs.overthewire.org
- ポート: 2220
- ユーザー: bandit<レベル番号>（例: bandit0, bandit1, ...）

接続コマンド例:
  ssh bandit0@bandit.labs.overthewire.org -p 2220

## 課題の目標

- Level 0 から Level 34（最終レベル）まで全問を解く
- 各レベルで次のレベルのパスワードを入手する
- Linux コマンド・ファイル操作・暗号・ネットワークなどの技術を習得する
